# 🧬 MedTrackInsight

**Search, Filter, and Benchmark Global Molecular Pipelines**

MedTrackInsight is a high-performance query console and benchmarking platform designed to empower biopharmaceutical R&D decisions. It provides critical researchers, investors, and clinical sponsors with high-resolution data, consolidating global pharmaceutical pipelines into a single, comprehensive dashboard. 

Access exhaustive clinical and chemical details across 250+ FDA & EMA pipelines, and seamlessly query candidates by drug name, indication, mechanism of action (MOA), and more.

## ✨ Key Features

### 🔍 Advanced Pipeline Prospector
* **Dynamic Search & Filtering:** Filter seamlessly by Therapy Area, Disease, Current Development Phase, Company/Sponsor, Biomarker/MOA, Product/Candidate, Biological Class, and Marketed/Off-Patent status.
* **45+ Data Dimensions:** Unlock comprehensive data columns including trial starts, sponsor organizations, molecular weight metrics, chemical formulas, and patent tables.
